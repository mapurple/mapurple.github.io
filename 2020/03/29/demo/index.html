Hexo搭建个人博客

目前使用的系统是Windows10 64位，所以这是基于window的Hexo博客搭建教程
1.安装Node.js

​ 打开Node.js官网：https://nodejs.org/en/

​

​

下载并安装Node.js，安装过程也是一直点“Next”下一步默认安装即可。

2.安装Git

​ Git其实是个程序员必备的工具，如果你没有安装，那么你一定是个假程序员（哈哈哈哈，开个玩笑）

​ 打开Git官网：https://git-scm.com/downloads

​ 选择Windows 64位下载

下载完成之后的安装过程也是一直下一步默认安装即可。

安装完成之后我们可以在开始菜单中看到

在这里需要用到的是Git Bash，这个也就相当于Linux中的终端，接下来的搭建步骤都需要用它来完成。

3.安装Hexo

​ 打开Git Bash

​ 首先查看一下node和npm是否安装成功

这里显示出版本号，就表示我们已经成功安装。

由于使用默认的npm下载较慢，所以这里我们可以使用npm安装一个淘宝的cnpm镜像源来提高下载速度。

使用cnpm全局安装Hexo

验证hexo是否安装成功

进入到这个新建好的文件夹

在这个文件夹下初始化博客

这里就已经搭建成功了，可以去文件夹下查看已经生成好的文件

接下来在浏览器输入：http://localhost:4000/ 就可以访问搭建好的博客

然后点击 Create repository

配置SSH key
